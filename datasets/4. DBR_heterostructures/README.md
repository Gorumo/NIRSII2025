## 📄 Описание данных: Спектры отражения распределенных брэгговских отражателей (РБО)

Данный репозиторий содержит спектральные данные, полученные для распределенных брэгговских отражателей (Distributed Bragg Reflectors, DBR), изготовленных из различных материалов и с разным числом пар слоев.

---

### 🧪 Контекст эксперимента

Цель эксперимента — изучить, как меняется спектр отражения РБО в зависимости от:
- Материалов, из которых изготовлены пары слоев (например, Al(92%)GaAs/GaAs)
- Количества пар слоев (например, x35)

---

### 📦 Структура данных

Формат: `Excel (.xlsx)`  
Каждый лист книги соответствует отдельной конфигурации РБО:  
`<материал> x<количество_пар_слоев>`  
Например: `Al(92%)GaAs-GaAs x35`, `DBR 850 nm x15` и т.п.

На каждом листе:

- Один или несколько образцов с данной конструкцией.
- Для каждого образца представлены три колонки:
  - `Wavelength` — длина волны в нм (общая для всех образцов)
  - `Signal` — измеренный спектр отражения в мВ
  - `Baseline` — темновые значения (фоновый сигнал)

---

### ⚙️ Преобразование данных

Для анализа и построения корректного спектра отражения необходимо:
```math
R(λ) = Signal(λ) - Baseline(λ)
```

---

### ❗ Особенности и ограничения

- Для некоторых листов/образцов отсутствует точная информация о материале или количестве слоев.
- Например: `DBR 850 nm x15` или `Al(92%)GaAs-GaAs` без указания количества пар.
- Уточнение по этим данным ведется и может быть добавлено позже.
