## 📄 Описание данных: Prediction of Critical Heat Flux (CHF)

В этом репозитории представлен набор данных, использованный в статье  
**"On the prediction of critical heat flux using a physics-informed machine learning-aided framework"**  
([Zhao et al., 2020, Applied Thermal Engineering](https://doi.org/10.1016/j.applthermaleng.2020.115190)).

Данные предназначены для предсказания **критического теплового потока (CHF)** с помощью моделей машинного обучения.

### 🧪 Контекст использования
Авторы сравнивают:
- **Физические модели** (традиционные инженерные подходы)
- **ML-модели** (машинное обучение)
- **Гибридные модели** (сочетание физики и ML)

В данной работе рассматриваются **только ML-подходы** для предсказания значения CHF.

---

### 📦 Структура набора данных

| Название столбца         | Описание                                       |
|--------------------------|------------------------------------------------|
| `id`                     | Уникальный идентификатор наблюдения            |
| `author`                 | Источник/автор экспериментальных данных        |
| `geometry`               | Геометрия канала (например, tube)              |
| `pressure [MPa]`         | Давление, МПа                                  |
| `mass_flux [kg/m2-s]`    | Массовый поток                                 |
| `x_e_out [-]`            | Выходная степень сухости                       |
| `D_e [mm]`               | Внешний диаметр                                |
| `D_h [mm]`               | Гидравлический диаметр                         |
| `length [mm]`            | Длина канала                                   |
| `chf_exp [MW/m2]`        | Экспериментально измеренное значение CHF       |

---

### 🔢 Размер и формат

- Формат: `CSV` с заголовками
- Количество наблюдений: **1865**
- Числовых признаков: **8**
- Целевая переменная: `chf_exp [MW/m2]`
