## 📄 Описание данных: Прогнозирование параметров уравнения Больцмана

В этом репозитории представлен набор данных, предназначенный для предсказания параметров, полученных в результате численного решения уравнения Больцмана, с использованием методов машинного обучения.

### 🧪 Контекст использования

Решение уравнения Больцмана зависит от ряда физических параметров. Для целей машинного обучения были зафиксированы второстепенные параметры, а варьировались лишь пять основных:

- Напряженность электрического поля `E` (в Тд)
- Температура возбуждения `T` (в К)
- Концентрация аргона `fraction_of_Ar`
- Концентрация гелия `fraction_of_He`
- Концентрация водорода `fraction_of_H2`

Задача — **предсказать среднюю энергию частиц `mean_Energy` и/или частоту импульсов `momentum_freq`** на основе этих входных данных.

---

### 📦 Структура набора данных

| Название столбца     | Описание                                                          |
|----------------------|-------------------------------------------------------------------|
| `E`                  | Электрическое поле, 100 значений от 1 до 500 Тд (1 Тд = 10⁻²¹ В·м²) |
| `T`                  | Температура возбуждения, 20 значений от 1000 до 6000 K            |
| `fraction_of_Ar`     | Доля аргона в смеси (от 0.0 до 1.0 с шагом 0.1)                   |
| `fraction_of_He`     | Доля гелия в смеси (от 0.0 до 1.0 с шагом 0.1)                    |
| `fraction_of_H2`     | Доля водорода в смеси (от 0.0 до 1.0 с шагом 0.1)                 |
| `mean_Energy`        | Средняя энергия частиц (таргет)                                   |
| `momentum_freq`      | Частота импульсов (таргет)                                        |

---

### 🔢 Размер и формат

- Формат: `CSV` с заголовками
- Тренировочный набор: **132 000 строк**
- Тестовый набор: **10 000 строк**

---

### ⚙️ Особенности параметрического пространства

Параметры заданы на равномерной сетке. Это позволяет:

- Использовать методы интерполяции в ячейке (например, KNN по ближайшим узлам)
- Сравнивать результаты модели с "априорной точностью" — например, средней ошибкой линейной интерполяции внутри ячейки сетки

---

### 🎯 Целевые переменные

Модели можно обучать на одну или обе переменные:

- `mean_Energy` — средняя энергия частиц
- `momentum_freq` — частота импульсов

- Исследование чувствительности таргетов к параметрам смеси и поля
