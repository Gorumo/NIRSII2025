## 📄 Описание данных: HIGGS (10 000 строк)

Этот репозиторий содержит подмножество из 10 000 строк оригинального набора данных [HIGGS](https://archive.ics.uci.edu/dataset/280/higgs), размещённого на UCI Machine Learning Repository. Данные были собраны в рамках задач физики высоких энергий и используются для различения событий, возникающих в результате распада бозона Хиггса, от фона стандартной модели.

### 📦 Источник
- **Полный набор данных**: 11 миллионов строк, доступен на UCI ML Repository  
- **Ссылка**: [HIGGS Dataset on UCI](https://archive.ics.uci.edu/dataset/280/higgs)

### 🧪 Цель
Классификация физических событий:
- `1` – событие связано с бозоном Хиггса (сигнал)
- `0` – событие не связано с бозоном Хиггса (фон)

### 🔢 Структура данных
Каждая строка содержит:
- **1 бинарная метка**: `Label` — 1 (сигнал) или 0 (фон)
- **28 числовых признаков**:
  - **7 восстановленных признаков** — получены на основе измерений
  - **21 высокоуровневый признак** — получены путём алгоритмической обработки

### 🧾 Формат
- CSV-файл с заголовками
- Разделитель: `,`
- Кол-во строк: 10 000
- Кол-во признаков: 29 (включая метку)

### 📁 Файл
- `HIGGS_10k_sample.csv` — первые 10 000 строк из полного датасета

📥 **Скачать файл:** [`HIGGS_10k_sample.csv`](./HIGGS_10k_sample.csv)
