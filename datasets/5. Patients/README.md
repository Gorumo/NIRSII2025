## 📄 Описание данных: Медицинские анализы пациентов

Этот репозиторий содержит код для анализа и предсказания возраста пациентов на основе их медицинских анализов.

### 📦 Источник
- **Набор данных**: Медицинские показатели пациентов
- **Файл данных**: combined_patients.csv

### 🧪 Цель
Регрессионный анализ для предсказания возраста пациентов на основе их медицинских показателей с использованием различных моделей:
- Линейная регрессия
- Случайный лес

### 🔢 Структура данных
Исходный набор данных содержит:
- **patient_id** - уникальный идентификатор пациента
- **age** - возраст пациента
- **date** - дата проведения анализа
- **Анализ** - тип проведенного анализа
- **Результат** - числовое значение результата анализа
- **Нижний референс** - нижняя граница нормы
- **Верхний референс** - верхняя граница нормы
- **status** - статус результата (нормальный=1, низкий=0, высокий=2)

### 🧾 Методология
- Создание табличного набора данных с анализами в качестве признаков
- Добавление производных признаков (отклонение от референсных значений)
- Разделение данных с учетом уникальных пациентов
- Обучение и сравнение моделей
- Статистический анализ ошибок предсказания

### 📁 Файлы
- `model.ipynb` — код анализа данных и моделирования
- `combined_patients.csv` — исходные данные о пациентах и их анализах

### 🛠 Требования
См. файл `requirements.txt` для списка необходимых библиотек.
