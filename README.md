# 📘 Проект №640104  
## Разработка интегрированной образовательно-исследовательской среды на базе платформы визуального программирования SMILE для решения задач в области естественных и инженерных наук

### 🏫 Университет ИТМО  
**Руководитель:** Колодезный Евгений Сергеевич  
**Платформа:** SMILE (визуальное программирование + машинное обучение)

---

## 🧭 Краткое описание

Проект направлен на создание среды, которая объединяет обучение и исследование в естественных и инженерных дисциплинах, используя визуальное программирование на базе платформы **SMILE**.  
Особое внимание уделяется интеграции задач обработки экспериментальных данных, моделирования и использования методов машинного обучения (ML).
